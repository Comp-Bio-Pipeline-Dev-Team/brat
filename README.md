# brat (**B**ulk **R**NA-seq **A**nalysis **T**ool)

An automated pipeline for bulk RNA sequencing analyses with known genomes (e.g. human, mouse, bacteria, virus).

## Getting started 

1. [Install `brat`](https://github.com/Comp-Bio-Pipeline-Dev-Team/brat/wiki/Installation-Methods) with your preferred method.
2. Prepare your [input files](https://github.com/Comp-Bio-Pipeline-Dev-Team/brat/wiki/Tutorials#required-input-files) and [metadata](https://github.com/Comp-Bio-Pipeline-Dev-Team/brat/wiki/Tutorials#configuring-required-csvs).
3. Choose the appropriate [execution](https://github.com/Comp-Bio-Pipeline-Dev-Team/brat/wiki/Tutorials#execution) and [deployment method](https://github.com/Comp-Bio-Pipeline-Dev-Team/brat/wiki/Tutorials#deployment-method) for your system and project.
4. Run the pipeline and explore your [report](https://github.com/Comp-Bio-Pipeline-Dev-Team/brat/wiki/Installation-Methods)!
